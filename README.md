# Re:Source — Official Website

Promotional website for **Re:Source**, a QR-based resource circulation platform that gives
leftover project materials and idle electronics a second life within the communities that
create them.

> Seoul Design Award 2026 entry.

**Live site:** https://jsonpassion.github.io/re-source/

This repository contains **only the marketing website** (static HTML/CSS). The iOS application
source code is maintained separately and is not published here.

## Structure

```
index.html        # single-page promotional site
assets/logo.png   # app icon artwork
.nojekyll         # serve files as-is on GitHub Pages
```

## Local preview

Open `index.html` in any browser, or serve the folder:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
